# 기능 목록 (Feature List)

이 문서는 태스크 관리 애플리케이션에 현재 구현된 기능들을 설명합니다.

## 1. 인증 (Authentication)

- **사용자 로그인**: 구글 OAuth(Google OAuth)를 연동하여 사용자를 인증합니다.
- **세션 관리**: 사용자 인증 데이터를 저장하여 로그인 상태를 유지합니다.
- **접근 제어**: 로그인한 사용자만 태스크 관리 기능을 이용할 수 있도록 제한합니다.

## 2. 할 일 관리 (Task Management)

### 할 일 생성 (Create Task)

- **새 항목 추가**: 모달 창을 통해 새로운 할 일을 생성할 수 있습니다.
- **입력 필드**:
  - 제목 (Title)
  - 설명 (Description)
  - 날짜/마감일 (Date)
  - 완료 여부 (Is Completed - 체크박스)
  - 중요 여부 (Is Important - 체크박스)

### 할 일 조회 (Read Tasks - Dashboard)

- **목록 표시**: 반응형 그리드 레이아웃으로 할 일 목록을 보여줍니다.
- **필터링 (Filtering)**:
  - **모든 항목 (All Items)**: 모든 할 일을 보여줍니다.
  - **완료된 항목 (Completed Items)**: 완료된 할 일만 필터링하여 보여줍니다.
  - **진행 중인 항목 (Proceeding Items)**: 아직 완료되지 않은 할 일만 보여줍니다.
  - **중요 항목 (Important Items)**: 중요 표시가 된 할 일만 보여줍니다.
- **상세 보기**: 읽기 전용 모달 창에서 할 일의 전체 내용을 확인할 수 있습니다.

### 할 일 수정 (Update Task)

- **내용 수정**: "수정(Update)" 모달을 통해 할 일의 모든 내용(제목, 설명, 날짜 등)을 변경할 수 있습니다.
- **빠른 완료 처리**: 카드 내의 버튼을 클릭하여 "완료(Completed)" 상태를 즉시 토글할 수 있습니다.

### 할 일 삭제 (Delete Task)

- **항목 삭제**: 시스템에서 할 일을 영구적으로 삭제할 수 있습니다.

## 3. UI/UX

- **반응형 디자인**: 다양한 화면 크기에 맞춰 레이아웃이 조정됩니다.
- **모달 인터페이스**: 페이지 이동 없이 현재 화면 위에서 할 일을 생성, 수정, 조회할 수 있습니다.
- **로딩 스켈레톤 (Loading Skeletons)**: 데이터를 불러오는 동안 뼈대(Skeleton) 애니메이션을 보여주어 사용자 경험을 향상시킵니다.
- **토스트 알림 (Toast Notifications)**: 생성, 수정, 삭제 등의 작업 성공/실패 여부를 실시간 팝업 메시지로 알려줍니다.
- **다크 모드**: 눈이 편안한 어두운 테마(Dark Theme)를 적용했습니다.
